# BMW


اDashboard بتعرض تحليل شامل لبيانات موديلات BMW من حيث:
 (Median Price)
 (AVG Mileage)
(AVG MPG) متوسط استهلاك الوقود
(Price By Year)
 مقارنة الأسعار بين الموديلات
 أهم ميزة في المشروع:
لما تفلتر على ايي موديل عربية فيهم تظهرلك صورة العربية و بيتم تحديث كل المؤشرات تلقائيًا، وده بيعكس تجربة تفاعلية أقرب للواقع
 المشروع بيوضح مهاراتي في:
تنظيف وتحويل البيانات باستخدام Power Query
بناء Data Model
كتابة معادلات DAX
تصميم Dashboard احترافي
إنشاء عناصر تفاعلية Dynamic
